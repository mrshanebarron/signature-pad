# Laravel Design Signature Pad

Signature drawing pad component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/signature-pad
```


## Usage

### Livewire Component

```blade
<livewire:ld-signature-pad />
```

### Blade Component

```blade
<x-ld-signature-pad />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-signature-pad-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-signature-pad-views
```
